## ✨ 主要功能

*   **逼真的3D交互**：通过鼠标拖拽，模拟真实转动。
*   **专业计时与统计**：毫秒级计时，自动计算Ao5, Ao12, TPS等数据。
*   **成绩分析**：提供成绩趋势图，并可将所有数据导出为CSV。
*   **公式工具**：一键生成打乱公式，并支持解法回放。
*   **自动求解**：在卡壳时可自动完成剩余步骤。
*   **PWA支持**：可“安装”到电脑或手机桌面，支持离线使用。
